# Individual Project - VR Historical World
### The prior project was based on Deepfake Detection and has now been changed to the creation of an educational video game. This project is now a historical educational video game that will be an explorable and interactive simulated world. This GitLab serves the purpose of storing all the scripts of features and asset references.  

>### E-Logbook Link: https://docs.google.com/document/d/1uU2kZqkrAnQ_Zo3aISDKNaQV0tWyBQxg9941YaPdqmc/edit?usp=sharing
